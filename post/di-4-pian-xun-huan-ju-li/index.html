第4篇循环举例

2020-10-13

源码:
1)/*
循环举例算1+3+5+7+9/1—10之间的奇数和
*/
#include<stdio.h>
int main(void)
{
int i;
int sum = 0;
for (i = 1; i < 10; i += 2)//这里不要加分号！！！切记！！！
{
sum = sum + i;
printf("哈哈哈哈哈哈哈哈哈\n");//可以通过这个知道循环的次数
}
printf("sum = %d\n", sum);
return 0;
}
